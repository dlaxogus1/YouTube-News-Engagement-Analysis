# 높은 관심은 높은 영향력을 의미하는가?

## 한국 뉴스 유튜브 채널의 조회수와 이용자 참여도 분석

### 프로젝트 소개

본 프로젝트는 한국의 주요 뉴스 유튜브 채널(SBS, KBS, MBC, YTN)을 대상으로 조회수와 이용자 참여도(좋아요, 댓글) 사이의 관계를 분석한 데이터 분석 프로젝트입니다.

'조회수가 높은 뉴스 영상은 실제로도 높은 이용자 참여를 이끌어내는가?'라는 질문에서 출발하여 YouTube Data API를 이용해 데이터를 수집하고, R을 활용하여 통계 분석을 수행하였습니다.

---

## 연구 질문

> 조회수로 나타나는 대중의 관심은 실제 이용자 참여로 이어지는가?

---

## 데이터 수집

- 대상 채널
  - SBS 뉴스
  - KBS 뉴스
  - MBC 뉴스
  - YTN 뉴스

- 수집 방법
  - YouTube Data API

- 수집 범위
  - 각 채널 최근 2,500개 영상
  - 조회수 50만 회 이상 영상만 분석

- 최종 분석 대상
  - 총 132개 영상

---

## 분석 방법

- R
- Pearson 상관분석
- 다중회귀분석
- 데이터 시각화(ggplot2)

분석 변수

- 조회수(Views)
- 좋아요 수(Likes)
- 댓글 수(Comments)

---

## 주요 결과

### 상관분석

| 변수 | 결과 |
|------|------|
| 조회수 ↔ 좋아요 | r = 0.377 (약한 양의 상관관계) |
| 조회수 ↔ 댓글 | 유의하지 않음 (p = 0.114) |
| 좋아요 ↔ 댓글 | r = 0.453 (중간 수준의 양의 상관관계) |

### 회귀분석

- 좋아요 수는 조회수와 유의한 관련성을 보였다.
- 댓글 수는 조회수와 유의한 관련성이 나타나지 않았다.
- R² = 0.144

---

## 결론

조회수가 높은 뉴스 영상이 반드시 높은 이용자 참여를 이끌어내는 것은 아니었다.

조회수는 대중의 관심을 보여주는 중요한 지표이지만, 이용자의 적극적인 참여를 설명하기에는 한계가 있었다.
